Falling Sands

A collaborative pixel sandbox on the Usernode blockchain. Draw with physics-simulated elements and watch them interact in real time.

  • Pick an element from the toolbar (tap the swatch to see all elements)
  • Draw on the canvas — strokes appear locally as a preview
  • Send your strokes (tap the check mark) to commit them on-chain as a transaction
  • Everyone runs the same deterministic simulation locally

Each stroke is stored as a transaction memo on the Usernode blockchain. Your device runs the physics engine (WASM) locally.

Tap anywhere outside this panel to close.
